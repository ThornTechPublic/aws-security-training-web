Module 1 Challenge: MFA

If you have a lot of time remaining, you can configure MFA (multi-factor authentication) on your IAM account. For AWS, this is typically done with Google Authenticator, a virtual MFA device. MFA is recommended for any IAM account with console access — especially if it's a privileged account.

Here's a summary of the things you need to do:

  • In IAM, select your username
  • Click on the Security credentials tab
  • Next to Assigned MFA device, click Manage
  • Select Virtual MFA device and click Continue
  • Download the Google Authenticator app on your mobile device
  • Follow the instructions in the wizard, which will include:
    • Scanning the QR code with your camera
    • Entering the MFA code
    • Entering the MFA code a second time, after the first one times out
  • Sign out of AWS, and sign in again using MFA
