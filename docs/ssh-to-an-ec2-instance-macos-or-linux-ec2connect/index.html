Appendix: SSH to an EC2 Instance (MacOS or Linux)

Before you begin

You'll need:

  • The EC2 instance public IP
  • The private key to your EC2 key pair

Instructions

Open Terminal. You can search for it using the Command-Space shortcut.

SSH into the EC2 instance's IP address

ssh -i key-pair-private-key.pem ec2-user@<ip-address>

You'll be presented with a warning, asking if you trust the server's host keys.

The authenticity of host '3.81.107.237 (3.81.107.237)' can't be established.
ECDSA key fingerprint is SHA256:GmTqAoYoFaMko9A23i+29elwtDrt9/y3++19c5/QbVo.
Are you sure you want to continue connecting (yes/no)

Type yes to proceed.

You should see the command prompt of the EC2 instance.
